Artificial Intelligence and Machine Learning 

This research introduces SmartPrice AI, an intelligent decision-support system designed to mitigate the challenges posed by dynamic pricing on e-commerce platforms like Amazon and Flipkart. Online retailers frequently adjust prices based on demand, seasonality, and stock levels, creating significant uncertainty for consumers regarding the optimal time to purchase. To address this, the system leverages automated web scraping to collect real-time data, including price history, discounts, and customer ratings.The core of the study involves a comparative analysis between traditional statistical methods and modern deep learning architectures. Specifically, a baseline ARIMA model is compared against a Long Short-Term Memory (LSTM) network. The experimental results demonstrate that the LSTM model is significantly more effective at capturing complex, non-linear patterns and long-term dependencies in sequential price data. It achieved a prediction accuracy of 92.7% and a remarkably low Mean Absolute Error (MAE) of 0.019, substantially outperformed the ARIMA model's 81.4% accuracy.Integrated as a Flask-based API and deployed via a browser extension, SmartPrice AI provides users with real-time "Buy Now" or "Wait" recommendations directly within their shopping interface. Furthermore, comparative testing with a Random Forest baseline confirmed that the LSTM’s specialized memory gates provide a superior edge in tracking the high volatility of e-commerce markets. Ultimately, this research proves that AI-driven predictive tools can empower consumers to achieve maximum discounts despite algorithmic market fluctuations
